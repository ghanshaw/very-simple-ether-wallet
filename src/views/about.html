About

The Very Simple Wallet is a distributed app and smart contract created to demonstrate the basics of working with the Ethereum blockchain. The wallet allows users to move funds between the account attached to the contract, and three demo accounts. I built this project in Solidity, using the Truffle framework. I also use Webpack to combine modules and Yarn to manage dependencies.

The project consists of two smart contracts. The first contract was written to manage the wallet. Another contract was created for generate and manage the demo accounts.
